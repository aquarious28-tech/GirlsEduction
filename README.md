# Girls' Education Website 🌍

This is a Class 9 project about **Girls' Education**.  
It highlights the importance of education for girls, global challenges, and local action plans.

## Pages
- **Home (index.html):** Introduction to the topic
- **Global Issue (global.html):** Challenges faced by girls worldwide
- **Local Action Plan (action.html):** How we can help locally

## Features
- Built with **HTML & CSS**
- Styled with a banner image and responsive design
- Hosted for free using **GitHub Pages**
